Installation

Stable release

To install simulai, run this command in your terminal:

$ pip install simulai

This is the preferred method to install simulai, as it will always install the most recent stable release.

If you don’t have pip installed, this Python installation guide can guide you through the process.

From sources

The sources for simulai can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/ltizzei/simulai

Or download the tarball:

$ curl -OJL https://github.com/ltizzei/simulai/tarball/master

Once you have a copy of the source, you can install it with:

$ python setup.py install
